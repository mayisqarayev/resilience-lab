# Resilience Lab

Engineering-focused Spring Boot project demonstrating production-grade retry and failure handling patterns.

## Overview

This project simulates unstable downstream services and implements resilience strategies such as:

- Retry with exponential backoff
- Failure classification (retriable vs non-retriable)
- Fallback mechanisms
- Observability with metrics and logs

## Tech Stack

- Java 21
- Spring Boot
- Resilience4j (planned)
- Micrometer (planned)
- Docker (planned)

## Goals

- Demonstrate real-world failure handling patterns
- Apply production-level retry strategies
- Provide testable and observable system behavior
